### Before you start
1. 
  - clone this repo
  - move to the repository directory
  - run ```npm install```
2.
  - create `.env` file
  - write your environment variable
  ```
  DBHOST=dbserver.students.cs.ubc.ca
  CWL=<your cwl id>
  DBPASSWORD=<letter 'a' followed by your student number>
  DB=sbp
  ```

### How to start app
- ```npm start```
